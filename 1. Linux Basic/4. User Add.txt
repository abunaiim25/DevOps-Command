--------------- Create/Add User ---------------

# useradd -c "This is amniuser" amniuser === Created on home
# grep amniuser /etc/passwd

# useradd -d /user1 user1 === Created on / ( / = location root directory)
  grep user1 /etc/passwd
  grep user1 /etc/shadow === (not working for root permission)
# useradd -e 25/08/2025 user2 === User created with expire_date
  grep user2 /etc/shadow
# useradd -f 15 user3 === User created with expire_duration
  grep user3 /etc/shadow
  
User created, user with group wise:

  id user6

If you specify -M, the home directory will not be created, only the user account will be created.
# useradd -M user9  
  tail -n 10 /etc/group
  id user9
  cat /etc/default/useradd
  tail -n 5 /etc/shadow

useradd --help
-c "Final User" === Full name / comment field
-d /user123 === home directory
-e 2025-08-24 === account expiration
-f 10 === inactive days
-g 60000 === primary group
-G abu,user1,user10,user14,user3 === secondary groups
-p 12345 === password
-u 50000 === UID
-w === exact word match
user15 === username
