# Kubernetes Bash Autocompletion Setup on Ubuntu

### 1. Install bash-completion

Make sure the bash-completion package is installed:

bash
sudo apt-get update
sudo apt-get install bash-completion -y



---

### 2. Enable bash-completion

Source the global bash-completion script in your bash configuration:

bash
echo "source /etc/bash_completion" >> ~/.bashrc
source ~/.bashrc



> ⚠ If running as root, replace ~/.bashrc with /root/.bashrc.
> 

---

### 3. Enable kubectl autocompletion

Add the following line to your bash configuration:

bash
echo "source <(kubectl completion bash)" >> ~/.bashrc
source ~/.bashrc



---

### 4. (Optional) Use a shorthand alias k for kubectl

Add the alias and autocompletion support:

bash
echo "alias k=kubectl" >> ~/.bashrc
echo "complete -o default -F __start_kubectl k" >> ~/.bashrc
source ~/.bashrc



Now you can type:

bash
k get [TAB]



instead of kubectl get.

---

### 5. Test autocompletion

Try typing:

bash
kubectl get [TAB]
kubectl describe [TAB]
kubectl apply -f [TAB]



If it shows suggestions, the setup is complete.

---

✅ *Result:* Kubectl commands now support full bash autocompletion.
